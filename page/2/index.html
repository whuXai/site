A4-Accessibility无障碍设计

发表于 2019-08-24 | 更新于 2019-09-04 | 分类于 UX 学习课程

好的无障碍设计是网站或应用程序成功的一个关键，不仅因为无障碍设计在许多国家是法律明文规定的，还要考虑到上百万、千万的各类不便用户，这个数字随着老龄化还在增加。不管出于道德和法律上的原因，还是为了让你的产品能够充分发挥潜能，都应该做好无障碍设计。
本章节将从各个角度了解无障碍设计，知道设计决策如何影响无障碍和可用性；应该遵循的无障碍的标准；如何将无障碍的纳入设计和开发过程，以及如何评估和验证。

有关Accessibility，有几个不错的系列：
首先是W3C的WAI及其解释说明的部分；
然后是WebAIM，斯坦福的SOAP，伯克利的总结；
或者看Google的开发者文档，IBM的规范，Material的指引和游戏方面的Guidelines。

IDF课程本质上也是抽取汇总了各个系列的部分内容。

A3-Affordance可供性

发表于 2019-08-23 | 更新于 2019-09-04 | 分类于 UX 学习课程

Affordance可供性，对于设计师来说是一个重要的概念，指的是对目标采取行动的可能性，例如电梯按钮可以按下去，椅子可以坐上去。
这个概念在20世纪80年代后期由Don Norman普及：用户应该能够快速和准确地识别设计所提供的行动内涵。因此，Affordance做的越好，就越有可能防止用户可能的失望或者沮丧。

我是从Hi-id的文章对可供性产生的兴趣。

本章节会介绍Affordance的理论，演进过程中的主要里程碑，Gibson、Norman、Gaver、Hartson等人的观点和理论，以及如何应用，使设计可以让用户采取自然地操作，无需犹豫地问自己“如果我这样做，会发生什么？”，为了“不要让用户思考”吧。

A1-用户体验

发表于 2019-08-21 | 更新于 2019-09-04 | 分类于 UX 学习课程

提出用户体验这个名词的Don Norman认为：用户体验是你如何认识这个世界，是你如何体验你的生活，是你如何享受服务，不只是某个软件、网站或APP，是一个更加全面的体系，覆盖多个不同的领域；理解这些不同的领域，在你从事用户体验工作时，就会知道如何用好相关工具。
