B4-移动端UX

发表于 2019-08-28 | 更新于 2019-09-04 | 分类于 UX 学习课程

2014年移动端设备的使用率已经超过桌面端，在移动设备上设计一个好的用户体验时，使用常规的设计方法是不够的，用户在移动设备上的理解能力要降低50%左右，这意味着内容、导航和可视化设计元素要比桌面端的直观个两倍，屏幕变小带来容错度的降低，设计师必须要掌握相应的新技能，用于创建易于使用的移动用户体验。

本章节基于Frank Spillers的经验分享，在经验主义的基础上的，结合业界研究和实践中提炼出来的确凿证据，学习如何理解、设计、评估和验证移动端的用户体验。
