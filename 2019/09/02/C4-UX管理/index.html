C4-UX管理

发表于 2019-09-02 | 更新于 2019-09-04 | 分类于 UX 学习课程

理解和授权UX团队，会使公司变得与众不同，通过更优化的用户体验，推动了业务增长，从UX的投入产出中获益。

本章节研究UX的ROI和商业目标如何关联，UX管理如何直接影响ROI的正面和负面；如何有效地雇佣、定位和授权UX团队，以及团队如何产生影响；UX的战略管理如何实施和推动。

一，从哪里开始？

1，从User Advocacy开始，两种思路，一种是对公司团队宣传UX，IDF有相关文章，也可以参考这几个tips，或者了解下SAP是如何做的；还有一种是对用户；
2，工作流程，除了设计流程也要建立适当的工作流程，参考案例【1】和【2】；
3，UX成熟度，评价你所在组织的UX成熟度，我们知道产品和技术都有所谓的成熟度，包括现在那条经常看到的Gartner曲线，UX也有各种各样的成熟度模型，Vetrov的写过一篇文章，NN/g也有对于成熟度的详细描述：【1】和【2】；
4，用户体验团队，参考UX Studio的这篇文章，或者UX Pin的小册子；也可以看看NN/g的课程一个人的UX团队。

5，什么是好的、差的团队管理者？参考文章【1】和【2】；

五，UX路线图和进程

第一部分，绘制你的UX战略图和理想的工作流程和成熟度
1，绘制你的UX战略图，参考Vetrov的电子书，Johnny Holland的文章，或者这类视频；
2，计划你的UX成熟度，可以参考World IA Day上的视频，或者是参考个Roadmap；

第二部分，团队管理和组织文化
1，作为一个Leader，你的角色很多样；
2，组建一个团队，可以参考手册，也有How-To类文章可以看；
3，组织文化有作用的3个关键，谈企业文化的文章很多，3个关键，4个关键，也许有很多关键；
4，建立你的团队：组织结构图，参考文章；
5，招聘注意事项：该做的和不该做的，参考文章；

六，持续增长与动力

第一部分，UX的实践和衡量
1，确保UX是可实践的，可衡量的，参考文章，实践案例；
2，确保UX团队有所表现，对应文章，参考案例；

第二部分，推动和激励策略
1，推动UX：已知的最佳方法，可以参考NN/g的文章和UXmatters的讨论；
2，激励UX发展的策略，理解产品战略和市场动态，关注增长相关的策略；

1，达到临界量：迹象/信号，对应文章，延伸参考；
