AR和VR设计

发表于 2019-09-03 | 更新于 2019-09-04 | 分类于 UX 学习课程

AR和VR是个很奇怪的领域，现在可能如Gartner曲线中爆发前的低谷，也可能是持续的低潮，苹果、微软和谷歌等巨头都有参与这个技术领域，但是还没有看到全民应用的场景。尽管有预测说到2025年，全球AR和VR的用户基数将达到4.43亿。
如果真的发生，那如何打造AR和VR的体验也会变得重要，3D空间的体验设计相对2D界面设计也许需要全新的方法。

本章节基于Frank Spillers的经验分享，了解3D体验设计所需的技能，在动态空间中，界面设计结合手势、动作、声音等元素，塑造AR和VR的用户体验。

C3-游戏化设计

发表于 2019-09-01 | 更新于 2019-09-04 | 分类于 UX 学习课程

游戏化是向现实世界或工作生产中融入游戏元素的过程，通过使产品或服务适合用户的生活，并以一种吸引人的方式达成，游戏化创造了独特的、富有挑战的体验，通过创造新的方式来建立长期合作并且可以扩展的关系，改变了商业模式，并且推动了客户和员工的忠诚度，把公司的业务适当游戏化即可快速尝鲜。

IDF的这个章节，基本上是Janaki Kumar的书 Gamification at Work: Designing Engaging Business Software，如何使用以玩家为中心的设计进行游戏化；管理、监控和衡量游戏化工作的影响，整理汇报游戏化的成果和收益。可以先看一篇表现她核心观念的文章。

我也看了Yu-Kai Chou的《游戏话实战》，两本书可以结合在一起了解游戏化设计。

C2-情感化设计

发表于 2019-08-31 | 更新于 2019-09-04 | 分类于 UX 学习课程

伟大的产品和优秀的产品真正的区别在于用户如何感受。每一种体验都有情感成分，使用产品也不例外。因此，在设计产品或网站时，应将情感融入其中。

本章节基于Don Norman的情感化设计做了扩展，包括：设计中的情感反应和影响它们的人为因素，如何建立情感与设计的关系，本能的、行为的和反思的设计之间的差异，以及如何鼓励积极的设计，当然基础的是马斯洛的需求金字塔和三位一体的脑结构模型。

C1-交互设计心理学

发表于 2019-08-30 | 更新于 2019-09-04 | 分类于 UX 学习课程

对所有设计师来说，深入理解用户心理是必须的。
尽管用户之间永远存在着许多个体差异，但我们的产品是通过共同的心理部分把用户聚合在一起，人类思维的限制和能力对于我们大家来说都是相同的。发展对这些认知限制和能力的理解，了解我们如何与真实和虚拟世界中的事物进行交互是交互设计和良好用户体验的关键。

B4-移动端UX

发表于 2019-08-28 | 更新于 2019-09-04 | 分类于 UX 学习课程

2014年移动端设备的使用率已经超过桌面端，在移动设备上设计一个好的用户体验时，使用常规的设计方法是不够的，用户在移动设备上的理解能力要降低50%左右，这意味着内容、导航和可视化设计元素要比桌面端的直观个两倍，屏幕变小带来容错度的降低，设计师必须要掌握相应的新技能，用于创建易于使用的移动用户体验。

本章节基于Frank Spillers的经验分享，在经验主义的基础上的，结合业界研究和实践中提炼出来的确凿证据，学习如何理解、设计、评估和验证移动端的用户体验。

B3-人机交互设计

发表于 2019-08-27 | 更新于 2019-09-04 | 分类于 UX 学习课程

我们希望和计算机之间的交互可以像和人对话一样直接明了，当然很多产品和服务都无法做到这点，有的可能是技术、成本限制，有的可以改善，涉及到人类心理学、情感化设计，还有特别的设计流程等等，而人机交互涵盖了这些改善人机关系的内容，帮助你理解计算机的目标用户，如何让相关产品和服务运行顺畅。

IDF这个课程基本上都是HCI专家Alan Dix的视频课程；他还写了本《人机交互》，不过被吐槽无聊和无趣。
如果真觉得无聊，也可以试试斯坦福的课程视频；

Bill Moggridge的 Designing Interactions 中文版《关键设计报告》对我的影响更多更积极，书+视频，五分推荐。
Jenifer Tidwell的 Designing Interfaces 中文版《界面设计模式》也值得参考。

B2-可用性

发表于 2019-08-26 | 更新于 2019-09-04 | 分类于 UX 学习课程

无数的研究已经表明可用性在产品选择中很重要，用户对此有所预期，这种日益增长的预期给设计师们寻找功能和形式之间的甜蜜点带来了更多的压力。
可用性也是用户增长、留存和满意度的关键，必须把它融入到设计之中。

我把IDF上Frank Spillers和William Hudson两位老师的课程综合在一个课程里面。
Frank是ExperienceDynamics.com的CEO，
William Hudson是一个提供咨询和培训服务的UCD专家，他的Guerrilla UCD是一个16*90分钟的系列在线课程，主题有关可用性、用户体验和团队敏捷。

本章节介绍业内一些最成功的设计师提出的最重要的概念、方法、最佳实践和理论，如何设计出可用性极好的产品，持续地进行可用性测试，评估基于用户的各种假设，验证是否背离用户。

B1-用户研究

本章节了解用户研究的方法和策略，知道分析用研结果的方法，让用研的结果可以有效推进设计，以及如何通过Workshops介绍用户画像、客户旅程地图，同客户、用户沟通。
